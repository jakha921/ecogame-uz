# ПРИЛОЖЕНИЕ Б. СКРИНШОТЫ ПРИЛОЖЕНИЯ

Данное приложение содержит описания скриншотов интерфейса приложения EcoGame, зафиксированных в production-среде (https://ecogame.fullfocus.dev).

## Б.1 Главное меню

**Рисунок Б.1 — Главное меню EcoGame (Desktop)**

Экран отображает: заголовок "EcoGame" с логотипом листа, 6 карточек режимов игры (Tez o'yin, Kategoriya, Kunlik topshiriq, Marafon, Chiqindi saralash, Ta'lim markazi), бейдж "Yangi!" на кнопке ежедневного задания. Цветовая схема: зелёный градиент (#16a34a). Навигационная панель содержит ссылки на лидерборд, образование и профиль.

**Рисунок Б.2 — Главное меню (Mobile, 390px)**

Тот же экран на iPhone 14. Карточки режимов отображаются в одну колонку. Навбар адаптирован для тач-взаимодействия: кнопки ≥ 44px.

## Б.2 Экран квиза

**Рисунок Б.3 — Экран квиза (вопрос)**

Показывает: прогресс-бар (3/10), текущий счёт (450 ball), streak-бейдж (×2.0 зелёный), SVG таймер-кольцо (зелёный, 18 секунд осталось), текст вопроса на узбекском, 4 кнопки ответа в сетке 2×2.

**Рисунок Б.4 — Экран квиза (правильный ответ)**

После клика: выбранная кнопка стала зелёной с иконкой ✅, остальные кнопки стали серыми/неактивными. Под вопросом раскрылась панель ExplanationPanel с зелёным фоном и текстом объяснения на узбекском.

**Рисунок Б.5 — Экран квиза (неправильный ответ)**

Выбранная кнопка стала красной ❌, правильный ответ подсвечен зелёным ✅. Таймер остановлен.

## Б.10 Административный интерфейс

**Рисунок Б.10 — Django Admin (список вопросов)**

Таблица вопросов с колонками: ID, текст (обрезанный), категория (AIR/WATER/SOIL/...), сложность, тип, активен. Фильтры справа: по категории, сложности, типу. Поиск по тексту вопроса. Стиль — django-unfold (современный Material-like дизайн).
