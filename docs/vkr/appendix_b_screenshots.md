# Приложение Б. Скриншоты приложения EcoGame

> Приложение содержит скриншоты всех ключевых экранов веб-приложения EcoGame,
> развёрнутого на домене **https://ecogame.fullfocus.dev**.

## Б.1. Главная страница (Bosh sahifa)

Главная страница отображает сетку из четырёх игровых уровней с карточками.
Каждая карточка содержит номер уровня, название на узбекском языке, краткое
описание и статус доступности (разблокирован / заблокирован). В верхней части
расположен раздел «Kunning ekologik fakti» с фактом дня, загруженным из API
(`GET /api/v1/education/facts/random/`).

*[Рисунок Б.1 — Главная страница EcoGame с четырьмя уровнями]*

## Б.2. Страница регистрации (Ro'yxatdan o'tish)

Форма регистрации содержит поля: «Foydalanuvchi nomi» (имя пользователя),
«Nik» (псевдоним), «Elektron pochta» (электронная почта), «Parol» (пароль)
и «Parolni tasdiqlang» (подтверждение пароля). Валидация реализована через
React Hook Form + Zod на стороне клиента и через DRF Serializer на сервере.

*[Рисунок Б.2 — Форма регистрации нового игрока]*

## Б.3. Игровой экран — уровень «Kichik hovli»

Основной игровой экран состоит из двух слоёв:
- **Phaser.js canvas** (800 × 600) с тайлмапом двора, интерактивными зонами
  (деревья, водоём, мусор), анимациями объектов и HUD-сценой;
- **React overlay** (абсолютное позиционирование) с кнопками паузы и выхода.

Панель HUD (верхняя часть canvas) отображает четыре индикатора экосистемы:
«Havo sifati» (воздух), «Suv tozaligi» (вода), «Tuproq holati» (почва),
«Biologik xilma-xillik» (биоразнообразие) — каждый от 0 до 100 единиц.

*[Рисунок Б.3 — Игровой экран уровня 1 с активными индикаторами экосистемы]*

## Б.4. Игровое действие — посадка дерева

При клике на интерактивную зону отображается контекстное меню доступных
экологических действий. После выбора действия запускается анимация (tween
роста дерева, эффект листьев), индикаторы обновляются, в правом углу всплывает
+N очков.

*[Рисунок Б.4 — Контекстное меню действий на зоне «посадка дерева»]*

## Б.5. Уведомление о достижении

При выполнении условия достижения (например, «Birinchi daraxt» — посадить
первое дерево) в нижней части экрана появляется toast-уведомление с иконкой
достижения и его названием на узбекском языке.

*[Рисунок Б.5 — Toast-уведомление о разблокированном достижении]*

## Б.6. Таблица лидеров (Yetakchilar)

Страница отображает топ-50 игроков в виде таблицы с колонками: ранг, аватар,
псевдоним, общий счёт, количество пройденных уровней. Текущий игрок
подсвечивается. Данные загружаются из `GET /api/v1/leaderboard/`.

*[Рисунок Б.6 — Таблица лидеров с топ-50 игроков]*

## Б.7. Образовательный раздел (Ta'lim)

Раздел «Ta'lim» содержит список образовательных статей с фильтрацией по
категориям: «Hammasi» (все), «O'simliklar» (флора), «Suv» (вода),
«Chiqindilar» (отходы), «Energiya» (энергия), «Hayvonlar» (животные).
Все статьи написаны на узбекском языке.

*[Рисунок Б.7 — Образовательный раздел с фильтрацией по категориям]*

## Б.8. Профиль игрока (Profil)

Страница профиля отображает: имя пользователя, псевдоним, дату регистрации,
общий счёт, количество пройденных уровней, а также сетку достижений
(разблокированные отмечены цветными иконками, заблокированные затемнены).

*[Рисунок Б.8 — Профиль игрока с достижениями]*

## Б.9. Административная панель Unfold

Панель администратора (`/admin/`) выполнена на базе библиотеки Unfold с
тёмной темой и адаптивным дизайном. Отображает все разделы: игроки, уровни,
экологические действия, сессии, прогресс, достижения, образовательный контент,
таблица лидеров.

*[Рисунок Б.9 — Административная панель Unfold с разделом «Уровни»]*

## Б.10. Деплой на ecogame.fullfocus.dev

HTTPS-сертификат выдан Let's Encrypt через Traefik certresolver. Приложение
доступно по адресу `https://ecogame.fullfocus.dev`. Docker Compose управляет
четырьмя контейнерами: postgres, backend, frontend, nginx.

*[Рисунок Б.10 — Браузер с открытым ecogame.fullfocus.dev и действующим SSL]*
