# Введение

## Актуальность темы

Экологические проблемы Узбекистана — загрязнение атмосферного воздуха (индекс AQI 120–160 в Ташкенте), катастрофа Аральского моря (потеря 92% водного объёма за 60 лет), неудовлетворительная система обращения с отходами (переработка менее 5% ТБО) — требуют системного подхода к формированию экологической культуры населения [1, 2]. Государственная программа «Yashil makon» (2021–2025) сосредоточена на озеленении, однако остро ощущается дефицит доступных образовательных инструментов для молодёжи [3].

Геймификация — применение игровых механик в образовательных целях — доказала свою эффективность в многочисленных исследованиях: мета-анализ Hamari et al. (2014) охватывает 24 эмпирических исследования и подтверждает статистически значимое повышение мотивации и усвоения материала [14]. Теория потока Чиксентмихайи (1990) объясняет, почему хорошо спроектированные игры удерживают внимание учащихся значительно дольше традиционных методов [5].

Вместе с тем анализ рынка экологических образовательных игр выявил критическую нишу: среди существующих решений (Eco, Toca Nature, SimEarth) ни одно не ориентировано на узбекскую аудиторию, не использует контекстуально значимые для Узбекистана экологические проблемы и не разработано на узбекском языке. Исследования Юнеско подтверждают, что обучение на родном языке повышает эффективность усвоения материала на 25–40% [6].

Таким образом, разработка экологической образовательной игры-симулятора на узбекском языке, адресованной школьникам и студентам Узбекистана, представляет собой актуальную научно-практическую задачу.

## Цель работы

Целью данной выпускной квалификационной работы является **разработка веб-приложения «EcoGame» — экологической игры-симулятора экосистемы на узбекском языке** для формирования экологической культуры у школьников и студентов Узбекистана на основе геймификации.

## Задачи исследования

Для достижения поставленной цели решаются следующие задачи:

1. Провести анализ экологической ситуации в Узбекистане и обосновать необходимость экологического образования через геймификацию;
2. Исследовать существующие аналоги экологических образовательных игр и выявить незанятую нишу для локализованного решения;
3. Провести сравнительный анализ технологий веб-разработки и обосновать выбор стека: Django REST Framework + React 19 + Phaser.js + PostgreSQL + Docker;
4. Спроектировать архитектуру системы, включая модели данных, REST API, игровую механику и пользовательский интерфейс;
5. Реализовать все компоненты системы: серверную часть (Django), клиентскую часть (React + Phaser.js), игровые механики симуляции экосистемы;
6. Развернуть приложение на домене ecogame.fullfocus.dev с использованием Docker Compose и Coolify;
7. Провести многоуровневое тестирование (87 автоматических тестов + E2E верификация).

## Объект и предмет исследования

**Объект исследования** — процесс экологического образования молодёжи Узбекистана через интерактивные цифровые технологии.

**Предмет исследования** — веб-приложение «EcoGame», реализующее механику экологического симулятора с элементами геймификации на узбекском языке.

## Методы исследования

- **Анализ и синтез**: изучение экологической ситуации Узбекистана, обзор литературы по геймификации;
- **Сравнительный анализ**: оценка технологических альтернатив по критериальным таблицам;
- **Моделирование**: проектирование ER-диаграмм, архитектурных схем, диаграмм состояний;
- **Программирование**: реализация системы с применением принципов TDD (разработка через тестирование);
- **Тестирование**: модульные, интеграционные и E2E тесты.

## Практическая значимость

Разработанная система может быть:
- Применена в образовательном процессе школ и колледжей Узбекистана в качестве дополнительного учебного материала по экологии;
- Использована как платформа для распространения актуальных экологических фактов и образовательных статей на узбекском языке;
- Расширена до мобильного приложения или мультиплеерного формата.

## Структура работы

Выпускная квалификационная работа состоит из введения, четырёх глав, заключения, списка литературы из 30 источников и четырёх приложений (А–Г). Общий объём — 72 страницы, 15 рисунков, 18 таблиц.

- **Глава 1** — аналитическая часть: экологические проблемы Узбекистана, обзор геймификации, анализ аналогов, сравнение технологий;
- **Глава 2** — проектная часть: архитектура системы, ER-диаграмма, REST API, игровая механика, UI/UX;
- **Глава 3** — реализация: описание ключевых компонентов с фрагментами кода, деплой;
- **Глава 4** — тестирование: 87 тестов, E2E верификация, анализ производительности.
