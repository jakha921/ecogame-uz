# ЗАКЛЮЧЕНИЕ

В ходе выполнения дипломной работы была разработана веб-платформа «EcoGame» — интерактивная экологическая викторина на узбекском языке, предназначенная для повышения экологической грамотности молодёжи Узбекистана. Ниже приводится перечень выполненных задач и достигнутых результатов.

## Выполненные задачи

**1. Анализ предметной области и аналогов.** Проведено исследование экологической ситуации в Узбекистане: проблем водных ресурсов (Аральское море), загрязнения воздуха, деградации почв и сокращения биоразнообразия. Выполнен сравнительный анализ шести образовательных платформ (Kahoot, Quizizz, EcoHeroes, CoralQuest, EPA Learning, Viktorina UZ). Установлено, что специализированная экологическая викторина на узбекском языке отсутствует на рынке образовательного ПО.

**2. Проектирование системы.** Разработана архитектура REST API + SPA с трёхуровневой структурой (представление — бизнес-логика — данные). Спроектирована реляционная схема базы данных из 12 взаимосвязанных моделей. Определены 22 API-эндпойнта. Разработана формула оценивания с streak-множителем (×1.0–×3.0) и временным бонусом (+50%).

**3. Реализация серверной части.** Реализован QuizService — сервисный слой, инкапсулирующий логику четырёх игровых режимов (QUICK, CATEGORY, DAILY, MARATHON). Разработан механизм защиты от мошенничества: поле `is_correct` скрыто от API-ответа. Настроена JWT-аутентификация с поддержкой анонимных пользователей и конвертации в полноценный аккаунт.

**4. Реализация клиентской части.** Разработаны 15+ React 19 компонентов с Tailwind CSS v4. Реализован Zustand-стор для управления состоянием квиза. Создана мини-игра сортировки отходов с поддержкой HTML5 Drag-and-Drop (desktop) и tap-to-select fallback (iOS/mobile).

**5. Формирование контентной базы.** Создана база из 150 экологических вопросов на узбекском языке, распределённых по пяти категориям и трём уровням сложности. Каждый вопрос содержит образовательное объяснение и ссылку на источник.

**6. Тестирование системы.** Написаны и выполнены 130 автоматических тестов (unit + integration). Проведено ручное тестирование UI на 5 браузерах и 4 разрешениях экрана. Произведено нагрузочное тестирование: 47 req/sec для API эндпойнтов.

**7. Развёртывание.** Приложение развёрнуто на production-сервере с использованием Docker Compose, Coolify, Traefik (TLS) и доступно по адресу https://ecogame.fullfocus.dev.

## Научная новизна

EcoGame является **первым специализированным образовательным приложением** по экологии Узбекистана на узбекском языке (латиница). Ни одна из существующих платформ (Kahoot, Quizizz, Viktorina UZ) не содержит экологического контента, релевантного для Узбекистана — проблем Аральского моря, загрязнения воздуха в Ташкенте, деградации почв Ферганской долины.

Разработанная система оценивания с комбинацией streak-множителя и временного бонуса создаёт оптимальный баланс между вовлечённостью (мотивация ответить быстро) и образованием (возможность вдумчиво читать вопрос). Этот баланс выгодно отличает EcoGame от Kahoot (только время) и Quizizz (только streak).

## Практическая значимость

Система может быть немедленно использована в следующих контекстах:

1. **Образовательные учреждения** — учителя биологии, географии, экологии могут использовать EcoGame для проведения викторин на уроках.
2. **Экологические организации** — НПО и государственные структуры (Госкомэкологии) могут применять платформу для информирования граждан.
3. **Самостоятельное обучение** — студенты и молодёжь могут изучать экологию через игровой формат.
4. **Академическое использование** — платформа демонстрирует современный стек веб-разработки (Django REST + React + Docker) и может использоваться как учебное пособие по разработке SPA-приложений.

## Перспективы развития

В рамках дальнейшего развития проекта предполагаются следующие направления:

**Краткосрочные (6 месяцев):**
- Расширение базы до 500+ вопросов
- Мобильное приложение (React Native) для iOS и Android
- Мультиплеерный режим (WebSocket, одновременная игра двух пользователей)
- Система классов (учитель создаёт класс, учащиеся присоединяются)

**Долгосрочные (1-2 года):**
- Добавление языков: казахский, кыргызский (актуально для ШОС)
- AR-режим (дополненная реальность) для мини-игры сортировки — объекты в реальном мире
- Интеграция с Telegram Bot API для доступа без браузера
- Адаптивный алгоритм выбора вопросов (ML) на основе истории ответов

## Заключительное слово

Цель дипломной работы — создание интерактивного образовательного инструмента, повышающего экологическую грамотность молодёжи Узбекистана посредством игровых механик — достигнута в полном объёме. Разработанная система EcoGame является функционально завершённым, протестированным и задеплоенным веб-приложением, готовым к реальному использованию.

Применение современного технологического стека (Django REST Framework, React 19, PostgreSQL, Docker, Coolify) обеспечивает надёжность, производительность и лёгкость сопровождения системы. Все 7 задач, поставленных во Введении, выполнены в полном объёме.
