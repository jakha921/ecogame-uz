## АННОТАЦИЯ

**Рузибаев Жахонгир Дилмуратович.** Разработка экологической игры по охране окружающей среды (на узбекском языке). — Дипломная работа. — Ташкент: Ташкентский университет информационных технологий, 2026. — 70 с.

Дипломная работа посвящена разработке веб-приложения «EcoGame» — геймифицированной экологической викторины на узбекском языке. Целью работы является создание интерактивного образовательного инструмента, повышающего экологическую грамотность молодёжи Узбекистана посредством игровых механик.

В ходе исследования проведён анализ существующих аналогов образовательных игр (Kahoot, Duolingo, EcoHeroes), обоснован выбор технологического стека (Django REST Framework, React 19, PostgreSQL, Docker) и разработана архитектура системы типа REST API + SPA. Реализован квиз-движок с четырьмя режимами игры (быстрый, по категориям, ежедневный, марафон), системой очков со стриком-множителем, шестью рангами, десятью достижениями и мини-игрой сортировки отходов. База вопросов содержит 150 экологических вопросов на узбекском языке по пяти тематическим категориям. Приложение развёрнуто в production-среде на сервере Coolify.

Результаты работы: функциональное веб-приложение доступно по адресу https://ecogame.fullfocus.dev, покрытие тестами составляет 136+ тестов, все модули прошли интеграционное тестирование.

**Ключевые слова:** геймификация, экологическое образование, викторина, узбекский язык, Django REST Framework, React, веб-приложение.

## ABSTRACT

**Ruzibayev Jahongir Dilmuratovich.** Development of an Environmental Game for Environmental Protection (in the Uzbek Language). — Bachelor's Thesis. — Tashkent: Tashkent University of Information Technologies, 2026. — 70 p.

This thesis is dedicated to the development of the web application «EcoGame» — a gamified environmental quiz in the Uzbek language. The objective of the work is to create an interactive educational tool that enhances the ecological literacy of Uzbek youth through game mechanics.

The research involved an analysis of existing educational game analogues (Kahoot, Duolingo, EcoHeroes), justification of the technology stack selection (Django REST Framework, React 19, PostgreSQL, Docker), and design of a REST API + SPA architecture. A quiz engine was implemented with four game modes (quick, category-based, daily, marathon), a scoring system with streak multipliers, six rank titles, ten achievements, and a waste-sorting mini-game. The question database contains 150 ecological questions in Uzbek across five thematic categories. The application is deployed in a production environment on a Coolify server.

Results: a functional web application is accessible at https://ecogame.fullfocus.dev, test coverage comprises 136+ tests, and all modules passed integration testing.

**Keywords:** gamification, environmental education, quiz, Uzbek language, Django REST Framework, React, web application.
